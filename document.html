Document

Each Malina Document semantically represents either XML or JSON document (file).

The name of the Document can include all characters that are allowed in the XML element name except colon “:”.

By default the document represents XML document. You can explicitly specify that document represents XML by adding extension “.xml” to the name of the document.

If document’s name end with “.json” then the document represents JSON document.

The max length of the document’s name with extension (“xml” or “json”) is 255. The max length of the document’s name without extension is 251 (255-4, 4 is the length of “.xml”).

The document’s definition start with “!” followed by document’s name and colon “:” that marks the start of the document’s body:

!PurchaseOrder:

OR

!PurchaseOrder2016_08_01.xml:

OR

!PurchaseOrder2016_08_01.json:
