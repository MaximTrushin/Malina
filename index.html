Overview

Malina is a markup language designed to be people-friendly and be semantically compatible with both XML and JSON formats.

The purpose of the language is to improve productivity of data editing by replacing machine-oriented XML and JSON formats with human-oriented Malina format.

The data defined in Malina format is not supposed to be used by any type of API but instead it has to be converted in XML and JSON formats and then used by applications.

Malina achieves the higher people-friendliness and allows higher productivity by using clean, intuitive and people-friendly syntax and rich code reuse features which are unique among all existing data-formats and markup-languages.

This language guide is a draft. It describes the current state of Malina markup language and is subject to change.
