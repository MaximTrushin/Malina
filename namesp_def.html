Namespace declaration

Namespaces can be declared in the Module, Document or Alias Definition.

Although JSON do not support XML namespaces they can be used in the Malina document to create JSON files.

Namespace declaration has the following syntax. #namespace_prefix=namespace

For example:

#xsi = http://www.w3.org/2001/XMLSchema-instance
#ipo = http://www.example.com/myipo
