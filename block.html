Block

In Malina, blocks represent any content that consists of one or more statements having a common parent statement. 

In the following example, the element billTo has a block with one attribute and five elements.

billTo:
    @xsi.type = "ipo:US-Address"
    ipo.name = Robert Smith
    ipo.street = 8 Oak Avenue
    ipo.city = Old Town
    ipo.state = AK
    ipo.zip = 95819

Indents define Malina’s document structure. To ensure that indent wasn’t added occasionally, block always starts with a colon ":" followed by the indented block of statements.
