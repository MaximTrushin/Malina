JSON Array

In Malina, JSON array is a block including only either array item or value.

All types of JSON values are supported including string, number and literals true, false and null

array:
    = string literal
    = 4839
    = true
    = false
    = null

Array item is an element with an empty name. The following is an example of the array with 3 objects.

bookstore:
    :
        category = cooking
        title = Everyday Italian
        author = Giada De Laurentiis
        year = 2005
        price = 30.00
    :
        category = children
        title = Harry Potter
        author = J K. Rowling
        year = 2005
        price = 29.99
    :
        category = web
